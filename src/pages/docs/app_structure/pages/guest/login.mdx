# Login
Where unauthenticated users sign in. Successful logins redirect the user to the authenticated
home view while unsuccessful ones display an error to the user.

Authenticated users cannot go to this page to avoid a single user having access to multiple accounts at once. 

Note: This page's layout was inspired from [Chatsimple's sign-in/registration page](https://app.chatsimple.ai/)!
