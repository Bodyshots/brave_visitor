# Update User Password

Allows a logged-in user to update their password. The user must provide their current password and a new password. Validation ensures the current password matches the logged-in user's credentials.

## Request Headers

- **Content-Type**: application/json  
- **X-CSRFToken**: The CSRF token of the user  

## Request Body

- **password** (string, required): The current password of the user.  
- **new_password** (string, required): The new password to update to.  

**Example**:  
```json
{
  "password": "currentPassword123",
  "new_password": "newPassword456"
}
```

## Responses

### 200
The user's password is successfully updated.  

**Example**:  
```json
{
  "message": "Password updated!"
}
```

### 400
The current password does not match the logged-in user's credentials.  

**Example**:  
```json
{
  "message": "Current password does not match"
}
```

### 500
An error occurred while updating the user's password.  

**Example**:  
```json
{
  "message": "Error updating user password",
  "error": "Some descriptive error message"
}
```
